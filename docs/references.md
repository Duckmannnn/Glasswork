# References and Credits

This rice was built from several references, experiments, and fixes.

It is not a direct copy of any single setup. The final version combines ideas from r/unixporn, KDE widgets, Zen Browser mods, and my own trial-and-error process.

## Personal Notes

The most useful lesson from this rice was that more customization does not always make the desktop better.

The final setup became cleaner when I stopped forcing transparency everywhere and let KDE, Better Blur DX, Transparent Zen, and Zen Internet do most of the visual work.
